Project Overview :
Seekers of Sand is a Rogue-Like game developed on PC. You embody a Seeker, a master of alchemy that uses its knowledge to erase the dissonance from the desert. Your goal is to progress in various areas and overcome challenges to reach the Twilight Queen, responsible for the spread of the dissonance in the desert.

On this project, I'm the main programmer that is responsible for all technical aspects of the game and all the code architecture. I work mostly on gameplay features like the character for example but I also work tools to help the production of the team. An example of that is the development of the behavior tree tools, a feature that is not available in Unity.
