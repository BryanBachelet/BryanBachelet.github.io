Lich Engine
Lich Engine is a game engine project that my team and I realised during ours second year of studying of game programming. The goal of this project was to create a funcitonnal engine from scratch with base features like have sound or manage animations. Ours intention with this project was to create a game engine that is design to create horror game. For this project, we were 4 peoples to work on it during 3 months:
  • Lenny Rabinne
  • Mathe Erbisti
  • Antoine Mordant
  • Bryan Bachelet
We have also implemented plenty of others features to have complete and functionnal engine. Let's make a list none exhaustif :
  • Physics
  • Editor Interface
  • Animations
  • Sounds
  • UI
  • SkyBox
  • Resources Manager
The main focus of our engine was the graphics quality to have the best immersion, something that is realy important for horror game. So this engine implements multiples graphics effects like :
  • PBR
  • Shadow for every type of light
  • Deferred Rendering
  • SSAO
  • HDR
On this project, I work on multiple systems. The main one is the engine Editor UI which I implemented scene hierachy window, file broswer or inspector window for example.For this system, I implemented reflexion in order to facilitate the Editor creation. The second system on which I work was the Scene system of the engine. I made the scene manager system, and the save system of them.
